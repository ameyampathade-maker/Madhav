One-time setup using JSONBin.io (free). Takes 2 minutes.

How to get the API Key:
1. Go to jsonbin.io → Sign Up (free)
2. Click your name (top-right) → API Keys
3. Copy the Master Key (starts with $2b$…)
4. Paste it below and click Create My Data Bin

☁️ Auto-Sync to Google Drive (Optional)
Set up one-time to auto-sync all backups to Google Drive. Takes 3 minutes.
Step 1: Download Google Drive for Desktop (free)
Step 2: In Google Drive → Create folder: Madhav_Backups
Step 3: Right-click folder → "Add shortcut to My Drive" → set to sync

Then download backups from this app → they auto-save to that folder → instant Google Drive backup! ☁️
